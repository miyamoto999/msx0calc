# msx0calc

MSX0 Stack用の電卓です。

font8.dat、font12.dat、font16.dat、calc.binはそのまま、
calc.basはSJIS、CRLFに変換してMSX0に持って行ってください。
(前のバージョンにあったstrbuf.binは必要なくなったので削除してください。)
